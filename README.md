A simple Java-based ATM simulation program built with Java Swing for a school project.

## Project Overview
This application demonstrates a basic Automated Teller Machine (ATM) interface where users can perform standard banking transactions. It features a graphical user interface (GUI) to simulate the experience of using a real ATM.

## Features
- **Basic Banking**: Cash withdrawals, deposits, and balance inquiries.
- **Transfers**: Simple simulation of transferring funds to different bank accounts.
- **Bill Payments**: Options to pay for utilities like water, electricity, and internet.
- **Mobile Top-up**: Functionality to load mobile credit.
- **Account Management**: Change PIN and view transaction history.
- **Receipts**: Displays a transaction summary after successful operations.
